# proyectoCriptografia

Este es un proyecto para la materia de Criptografia de la facultad de ingenieria

    El programa "llavesrsa.py" se encargara de crear una llave privada y otra publica en el directorio donde este ubicado, almacenandolas en un archivo .pem
    El programa "encriptadorsa.py" se encargara de solicitar al usuario un mensaje, el cual sera encriptado y almacenado en un archivo .txt
    El programa "desencritarsa.py" se encargara de desencritpar el mensaje, utlizando las llaves generadas.
